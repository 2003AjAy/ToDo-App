# TaskFlow - Todo App

**TaskFlow** is a simple, responsive Todo app built with HTML, CSS, and Vanilla JavaScript. It helps users manage tasks across three stages: **Todo**, **Completed**, and **Archived**.

## 🚀 Features

- ✅ Age verification on landing page (must be >10 years)
- ✅ LocalStorage-based user registration
- ✅ User profile display with avatar using UI Avatars API
- ✅ Add, complete, archive, and restore tasks
- ✅ Display last modified timestamp
- ✅ Fetch dummy todos from DummyJSON API on first load
- ✅ Fully responsive design
- ✅ Sign-out functionality

## 🔧 Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- UI Avatars API
- DummyJSON API
- LocalStorage (for persistence)
